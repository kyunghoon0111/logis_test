# 물류관리사 기출문제 앱 (설치형 PWA)

물류관리사 **6개년(24~29회) 기출문제 1,200문항**을 실제 시험처럼 풀고, 채점받고,
**전 문항 해설**로 복습하는 태블릿용 학습 앱입니다.

## 실행 방법

### 1) PC에서 실행
프로젝트 폴더의 **`물류관리사_기출앱_실행.bat`** 을 더블클릭하세요.
(파이썬이 설치돼 있어야 합니다. 브라우저가 자동으로 열립니다.)

수동 실행:
```
cd app
python -m http.server 8137
```
브라우저에서 `http://localhost:8137/` 접속.

### 2) 태블릿에서 사용 (같은 와이파이)
1. PC에서 위 `.bat`을 실행하면 창에 `http://192.168.x.x:8137/` 형태의 주소가 표시됩니다.
2. 태블릿 브라우저(Chrome/Safari)에서 그 주소로 접속합니다.
3. 브라우저 메뉴 → **"홈 화면에 추가"** 를 누르면 앱처럼 설치됩니다.
   - 한 번 열어두면 서비스워커가 데이터를 캐시하여 **오프라인에서도** 동작합니다.

## 주요 기능
- **전과목 모의고사**: 5과목 200문제 · 자동 채점 · 합격 판정(평균 60점↑, 과목별 40점↑ 과락)
- **과목별 풀기**: 한 과목 40문제 집중
- **해설**: 정답 근거 + 오답 이유 + 실전 팁 + 관련 주제 (1,200문항 전부)
- **오답노트**: 틀린 문제 자동 수집 → 다시 풀기 / 해설 보기
- **즐겨찾기**, **진도 현황**, **다크 모드**, **진행중 시험 이어풀기**
- 복수정답/전항정답 기출도 정답 인정 처리

## 폴더 구조
```
app/
├── index.html              앱 진입점
├── css/style.css           스타일
├── js/app.js               앱 로직
├── data/exams.json         6개년 문제+해설 통합 데이터
├── manifest.webmanifest    PWA 설치 정보
├── sw.js                   서비스워커(오프라인)
└── icons/                  앱 아이콘
```

## 데이터 재생성
원본 `*/exam_structured.json` 을 수정한 뒤 데이터를 다시 만들려면:
```
python _build/build_data.py
```
