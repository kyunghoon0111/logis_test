# 기출문제 해설 추가 프롬프트

아래 프롬프트를 새 Claude 세션에서 사용하세요. 한 세션당 1개 연도씩 처리하는 것을 권장합니다.

---

## 프롬프트 (복사하여 사용)

```
물류관리사 기출문제에 해설을 추가하는 작업을 해주세요.

## 작업 대상
`c:\Users\김경훈\Downloads\물류관리사\{연도}년도 제{회차}회 물류관리사 기출문제\exam_structured.json`

## 현재 상태
- 5과목 × 40문제 = 200문제 (구조화 완료)
- 각 문제에 question, options, answer, answerLabel은 있지만 **explanation이 없음**

## 작업 내용
각 문제에 다음 필드를 추가해주세요:

1. **"explanation"**: 왜 해당 선지가 정답인지 + 오답 선지가 왜 틀린지 설명 (2~4문장)
2. **"examTip"**: 시험에서 이 유형을 만났을 때의 풀이 팁 (1~2문장)
3. **"relatedTopic"**: 관련 학습 주제 (예: "물류의 기능별 분류", "EOQ 계산")

## JSON 구조 (각 문제에 3개 필드 추가)
수정 전:
{
  "no": 1,
  "question": "...",
  "options": ["...", "..."],
  "answer": 4,
  "answerLabel": "④"
}

수정 후:
{
  "no": 1,
  "question": "...",
  "options": ["...", "..."],
  "answer": 4,
  "answerLabel": "④",
  "explanation": "정답은 ④이다. ...(해설)...",
  "examTip": "...(풀이 팁)...",
  "relatedTopic": "관련 주제명"
}

## 중요 주의사항
1. 해설은 한국어로 작성
2. 정답뿐 아니라 주요 오답 선지도 왜 틀린지 설명
3. 계산문제는 풀이 과정을 단계별로 제시
4. 법규 문제는 근거 조문(제X조)을 명시
5. 2026년 현행법 기준으로 해설 작성 (안전운임제 재도입 등 최신 개정 반영)
6. 단, 출제 당시 법률과 현행법이 다른 경우 "※ 현행법 기준 변경사항: ..."으로 부기
7. JSON 문법이 깨지지 않도록 주의 (큰따옴표 이스케이프 등)
8. 한 과목씩 순서대로 처리 (물류관리론 → 화물운송론 → 국제물류론 → 보관하역론 → 물류관련법규)

## 참고 자료
- 개념 데이터: `물관_데이터/` 폴더의 각 과목 concepts.json
- 이 파일들에 상세한 개념 설명, 비교표, 공식이 있으므로 해설 작성 시 참조

먼저 exam_structured.json을 읽고, 물류관리론 40문제부터 해설을 추가해주세요.
```
